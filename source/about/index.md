## 职业规划

{% endnote %}
✨✨✨

好的职业规划会清晰自己在未来要走的道路。如果只是拍脑袋地想着:"今天学点这个吧"，那将很难构筑牢固的知识体系。只会是松散的小平层。

列举了我目前拥有的技术栈：

1、熟悉大模型应用开发流程，具备 Prompt Engineering、Function Calling 与 RAG 实践经验，能够完成知识库问答与 Agent 应用开发
2、熟悉 Vue2 前端工程化实践，具备 AI 应用前端开发能力，能够实现基于 SSE 的流式对话、ChatUI 交互及大模型实时输出展示
3、熟悉 Dify Agent 编排与工作流设计，理解 ReAct 思想及工具调用机制，具备多工具协同与任务路由设计能力
4、了解大模型应用工程体系，熟悉 FastAPI 服务化开发，了解 vLLM 推理加速与 RAGAS 评估方法，具备 LoRA 参数高效微调及 RAG 核心链路（Embedding、向量检索、重排序）理解
