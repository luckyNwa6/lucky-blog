# Typora 常用语法

- **加粗**

```
**粗体** 或 __粗体__
```

选中文字 Ctrl+B **加粗**

- _倾斜_

```
*斜体* 或 _斜体_
```

选中文字 Ctrl+I 倾斜

- **_加粗斜体_**

  ```
  ***加粗斜体***
  ```

- **图片展示**

直接拖图片进来

```
![]() 小括号里面是图片的链接可以是图床里面的地址
![图片1](/1.jpg '这是描述内联的')
![](https://imgs.luckynwa.top/blog/loading.gif) 效果如下
```
