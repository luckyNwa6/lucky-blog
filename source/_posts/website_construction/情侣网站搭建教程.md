```bash
docker compose up -d
```

## 修改版权

修改文件：`/wwwroot/usr/themes/Brave/base/` 下的 `foot.php`

**原代码**：

```php
<p class="h6"> Powered by <a href="http://typecho.org" target="_blank">Typecho</a> ※ Theme is <a href="https://blog.zwying.com" target="_blank">Brave</a></p>
```

**改为**：

```php
<p class="h6"> Powered by <a href="https://luckynwa.top" target="_blank">luckyNwa</a></p>
```

## 方式2

### 服务器选择

推荐使用 **星辰云** 轻量云主机，性价比超高

- 官网：https://starxn.com/
- 价格：**2块钱一个月**
- 管理面板：https://aspanel.starzv.com/vhost/#/home/

### 域名配置

需要域名（二级域名即可），要做域名解析

1. 先腾讯云域名解析：https://console.cloud.tencent.com/cns
2. 然后绑定在自己的轻量云主机上面

### 环境要求

| 组件     | 版本                 |
| -------- | -------------------- |
| 宝塔面板 | 最新版               |
| Nginx    | 1.22.1（端口改为80） |
| MySQL    | 5.6                  |
| PHP      | 7.2.33               |

然后访问 `http://域名/` 进入 nginx 页面，加上 index.html 即可正常访问。

### 宝塔操作流程

```
宝塔 → 网站 → 添加站点 → 域名（ftp和数据库也要勾选）→ 双击根目录 → typecho 压缩包上传并解压
```

**上传目录无法写入？**

手动将安装目录下的 `/usr/uploads` 目录权限设置为可写：

```bash
cd /www
chattr -ia wwwroot
chmod -R 777 wwwroot
```

## 主题配置

将主题压缩包完整上传到服务器上 Typecho 的 `/usr/themes/` 文件夹内，解压，然后到 **Typecho 后台 → 控制台 → 外观 → 启用主题**
